科学上网指南

发表于 2016-10-08   |   分类于 干货分享   |     |   阅读次数

科学上网的办法有 N 多种，鉴于大多数人对偏专业性的手段比较无感，这里推荐的方法是相对最为方便、且高效的，本人亲测。网上也很有多其他办法,如果对本文里提到的方法不感兴趣，建议出门知乎一下，技术大神也请勿喷。

Windows 系统的电脑、OSX 10.8 系统及以上苹果电脑、 Android 系统手机

  • 推荐 Lantern （蓝灯）作为出墙工具，曾经我在知乎上以及个人博客里推介过，用过的网友大都表示不错，当然我自己也一直在用。其分为免费版以及专业版，两者的区别是：免费版每个月会有 700M 左右的高速流量，用完之后会变成普通的流量，也可以 FQ，但是网速可能会不太稳定；而专业版则是按年进行付费，付费之后不限流量，可以一直享用其带来的高速流量服务。

  • 至于选择免费版还是专业版，全靠个人喜好。如果是个老司机，我推荐还是可以试试专业版，比较开通之后，你除了可以上 Tumblr 外，很多网站也都可以畅通无阻的，到时候想干嘛就干嘛。

  • 鉴于直接放 Lantern 的下载地址，容易被和谐掉，所以放到了个人微信号中，大家可以搜索微信号：小正说事（xikexuezhang）(或者在文章末尾，选择扫描二维码关注)，然后回复关键词 lantern 即可获取。

iPhone 系统手机

  • Netfits 云墙 作为出墙工具，当然其也有 Android 版本的软件 云墙 - 玩的就是速度，你可以根据自己的需要下载使用，新用户立下载『云墙』，即可获赠 1GB 免费流量，这点亲测，确实可以。

  • 当免费流量用完之后，则需要付费才能够进一步 FQ 才行。

  • 在 Apple Store 的下载地址为：https://itunes.apple.com/cn/app/id1072676126 当然也可以直接搜索 Netfits 获取。

（在 iPhone 手机上的 VPN 软件多如牛毛，鉴于本人能力和精力有限，只能推荐自己亲自己用过且效果也不错的软件，如果各位大神有好的推荐，也欢迎在评论里留言分享。）

一根鱼刺，让我第一次走进急诊室

发表于 2016-07-15   |   分类于 随笔   |     |   阅读次数

两天前我还在知乎专栏里更新了这样的一篇文章《一名吃鱼爱好者 做的一些盘点 》，可谁曾想一天后，也就是昨天自己却因为晚上吃鱼被鱼刺卡住，自己尝试了各种办法无果之后，第一次进了急诊室。

高兴的去吃鱼 意外的被鱼坑

作为一个从小就爱吃鱼的人，从小到大吃鱼无数，距离上一次吃鱼被鱼刺卡住还是十多年前的事情，可能正是因为作为一名吃鱼的老司机过于的自信，觉得自己轻车熟路才导致的吧。

说实话，刚感受到被鱼刺卡住的时候，我用尽了各种办法，喝醋、吃馒头，以及呕吐，但效果均无，一开始那几分钟我觉得自己肯定能搞定这事儿，可办法用尽之后，我却整个人都变懵了，堪比世界末日的来临：一方面被鱼刺卡住之后，咽喉非常的疼，自己尽可能的靠喝水和咽口水来减轻疼痛感；另一方面则觉得随时都可能面临死亡之类的，非常的无助，而又迫切的想把异物取出。
